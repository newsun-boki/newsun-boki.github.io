数据预处理

数据的无量纲化

一般来说，当我们将数据导入模型的时候，无量纲化的可以帮我们去除量纲对模型的影响（决策树和随机森林不需要这样做，它可以处理大多数数据）
一般来说线性的无量纲化包括去中心化和缩放处理，中心化就是将原本的数据通过加减一个固定值使他进行平移，缩放就是对数据乘除一个固定值使其处于某一种范围之中，例如取对数。
